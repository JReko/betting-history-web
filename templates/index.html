How to Use BetTracker

1. Create a Bet

To create a new bet, click the "Create a New Bet" button in the top section. Fill in the required details like event date, stake, potential winnings, and the line, then submit to save the bet.

2. Edit a Bet

You can edit an existing bet by navigating to the Bets page. Find the bet you'd like to edit, click the edit button, and update any fields you need. Be sure to save your changes when you're done.

3. Assign Cappers

Each bet can be assigned to a capper, who provides betting tips. When entering a new bet, you'll have the option to enter the capper attached to the bet. There's also "Assign Cappers" feature to assign cappers in batch, this is mainly used when bets are imported and not created manually. Cappers help organize bets based on who made the pick and track how profitable they are.
